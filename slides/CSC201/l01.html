Number systems

The positional number system

A number $n$ expressed in a base $r$ system has coefficients multiplied by powers of $r$.

$n = a_n.r^{n-1} + a_{n-1}.r^{n-2} + … + a_2.r + a_1 + $

$a_{-1}.r^{-1} + a_{-2}.r^{-2} … + a_{-n}.r^{-n}$,

where $a$ ranges from 0 to $r$-1.

Example - Decimal numbers (base 10)

$(123.45)_{10} = 1 \times 10^2 + 2 \times 10 + 3 + 4 \times 10^{-1} + 5 \times 10^{-2}$

A few other examples

$(210.2)_5 =2 \times 5^2 + 1 \times 5 + 0 + 2 \times 5^{-1} = 55.4$ $(127.4)_8 =1 \times 8^2 + 2 \times 8 + 7 + 4 \times 8^{-1} = 87.5$

Example - Binary numbers (base 2)

$(10110)_2 =1\times 2^4 + 1\times 2^2 + 1\times 2 = 22$

Decimal to binary conversion

Whole numbers

Given a decimal number d, repeatedly divide d by 2 and save the remainders from right to left.

Keep going till no more division is possible.

Example: $(17)_{10}$

Fractions

Given a decimal fraction d, repeatedly multiply it by 2 and save the integer part of the result from right to left.

Example: $(.375)_{10}$

Mixed fractions

Separate the integer and fraction and work on them separately.

Example: $(41.75)_{10}$

Decimal to octal conversion

Decimal to octal conversion works the same way -

  • for integers, just keep on dividing by 8
  • for fractions, keep on myltiplying by 8

Example - $(175)_{10}$

Example - $(.27)_{10}$

Octal to binary conversion

Change each octal digit to its binary equivalent.

Example: 75

 7   5
 |   |
111 101

$(75)_8 = (111 101)_2$

Binary to Octal conversion

Just reverse the previous method

1001.0111 -> 001 001.011 100
              |   |   |   |
              1   1 . 3   4

$(1001.0111)_2 = (11.34)_8$

Hexadecimal numbers

Writing long binary numbers becomes tedious and prone to error. A group of four bits represents one of $2^4 = 16$ possibilities.

Hence, it is sometimes more convenient to work in base 16, called hexadecimal.

Hexadecimal numbers use the digits 0 to 9 along with the letters A to F,

Hexadecimal to binary conversion

8AC -> 8    A    C
      1000 1010 1100

$(8AC)_{16} = (1000 1010 1100)_2$

Hexadecimal to decimal

$A2 = 10 \times 16 + 2$

Decimal to hexadecimal

  • for integers, just keep on dividing by 16
  • for fractions, keep on myltiplying by 16
