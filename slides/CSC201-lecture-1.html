How does a computer really work?

The Three Big Ideas

  • Represent all data with discrete values

  • Store and process discrete values in the binary number system

  • Use voltage levels to represent, and electronic switches to process binary data

Abstraction

Hiding details when they are not important

Hierarchy - A system divided into modules and submodules

Modularity - Having well-defined functions and interfaces

Regularity - Encouraging uniformity, so modules can be easily reused

The digital abstraction

Most physical variables are continuous

  • Voltage on a wire
  • Frequency of an oscillation
  • Position of a mass

Digital abstraction considers discrete subset of values

The analytical engine

  • Designed by Charles Babbage from 1834 – 1871
  • Considered to be the first digital computer
  • Built from mechanical gears, where each gear represented a discrete value (0-9)

Binary values

Digital abstraction considers discrete subset of values

Two discrete values - 1’s and 0’s

  • 1 = TRUE = HIGH
  • 0 = FALSE = LOW

How to represent 1 and 0:

  • voltage levels, rotating gears, fluid levels, etc.

The positional number system

A number $n$ expressed in a base $r$ system has coefficients multiplied by powers of $r$.

$n = a_n.r^{n-1} + a_{n-1}.r^{n-2} + … + a_2.r + a_1 + $

$a_{-1}.r^{-1} + a_{-2}.r^{-2} … + a_{-n}.r^{-n}$,

where $a$ ranges from 0 to $r$-1.

Example - Decimal numbers (base 10)

$(123.45)_{10} = 1 \times 10^2 + 2 \times 10 + 3 + 4 \times 10^{-1} + 5 \times 10^{-2}$

A few other examples

$(210.2)_5 =2 \times 5^2 + 1 \times 5 + 0 + 2 \times 5^{-1} = 55.4$ $(127.4)_8 =1 \times 8^2 + 2 \times 8 + 7 + 4 \times 8^{-1} = 87.5$

Converting Whole numbers to binary

Given a decimal number d, repeatedly divide d by 2 and save the remainders from right to left. Keep going till you reach 0.

Example: $53_{10}$

53/2 = 26 R1

26/2 = 13 R0

13/2 = 6 R1

6/2 = 3 R0

3/2 = 1 R1

1/2 = 0 R1

Fractions

Given a decimal fraction d, repeatedly multiply the decimal part by 2 and save the integer part of the result from left to right. Keep going till the decimal part becomes 0.
