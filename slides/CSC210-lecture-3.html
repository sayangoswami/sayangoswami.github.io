Loop Invariants

  • Common method of showing correctness

  • An assertion that we demonstrate to be true each time we start a loop iteration

  • we have to show three things about a loop invariant:

    • Initialization: It is true before the first iteration of the loop.

    • Maintenance: If it is true before an iteration of the loop, it remains true before the next iteration.

    • Termination: The loop terminates, and when it does, the loop invariant, along with the reason that the loop terminated, gives us a useful property.

Loop invariant of linear search

def linear_search(A, x):
   for i in range(len(A)):
      if x == A[i]:
         return i
   return -1

Prove: the above algorithm correctly finds x in A.

Invariant: At the start of each iteration of line 2, if x is present in A, then it is present in the subarray from A[i] through A[n-1].

Initialization: Trivially true

Maintenance: For iteration i, if x is present, it must be in A[i…n-1]. If we are at iteration i+1, then A[i] $\neq$ x, and therefore x if present must be in A[i+1…n-1].

Termination: The loop must terminate either due to line 4 or 5. It terminates at line 4 only of we have found x.

For line 5 use contrapositive: if x is not present in the subarray from A[i] through A[n-1], then it is not present in the array A.

When i = n, the subarray is empty, so it cant have x.

Recursion

Solve a problem by solving smaller instances of the same problem.

Example: $n! = n.(n-1).(n-2)\dots 3.2.1$

Note that $(n-1)! = (n-1).(n-2)\dots 3.2.1$

Therefore $n! = n \times (n-1)!$

For recursion to work:

  • First, there must be one or more base cases, where we compute the solution directly without recursion.
    • Base case for factorial?
  • Second, each recursive call of the procedure must be on a smaller instance of the same problem that will eventually reach a base case.

Recursive linear search

def recursive_linear_search(A, x, i=0):
   if i >= len(A):
      return -1
   elif A[i] == x:
      return i
   else:
      return recursive_linear_search(A, x, i+1)

recursive_linear_search(A, x)

Binary search

def binary_search(A, x):
   p = 0
   r = len(A)-1
   while p <= r:
      q = int((p+r)/2)
      if A[q] == x:
         return q
      elif A[q] > x:
         r = q - 1
      else:
         p = q + 1

Recursive binary search

def rec_binary_search(A, p, r, x):
   if p >= r:
      return -1
   else:
      q = int((p+r)/2)
      if A[q] == x:
         return q
      elif A[q] > x:
         return rec_binary_search(A, p, q-1, x)
      else:
         return rec_binary_search(A, q+1, r, x)

rec_binary_search(A, 0, len(A)-1, x)

def selection_sort(A):
   for i in range(len(A)):
      si = find_smallest_index(A, i)
      swap(A, i, si)

def find_smallest_index(A, i):
   si = i
   for i in range(i, len(A)):
      if A[i] < A[si]:
         si = i
   return si

def swap(A, i, si):
   temp = A[i]
   A[i] = A[si]
   A[si] = temp

def insertion_sort(A):
   for i in range(1, len(A)):
      key = A[i]
      j = i-1
      while j >= 0 and A[j] > key:
         A[j+1] = A[j]
         j = j-1
      A[j] = key
